dwm

suckless.org's dwm with a built-in MPD client, and more

View on GitHub Download .zip Download .tar.gz

This software is a X11 window manager based on suckless.org's dwm 6.0, adding a client for the music player daemon and various other gimmicks.

Patches

The code incorporates the following patches by various authors which are or at some point in time have been listed on dwm.suckless.org.

  • pango
  • pertag
  • savefloats
  • pushstack
  • cycletag
  • bstack layout
  • deck layout
  • Cherrypick from upstream git: Use XkbKeycodeToKeysym() instead of XKeycodeToKeysym()

These are changes made by me. A clean patchset against dwm 6.0 doesn't exist; this mod is a take-it-or-leave-it thing. View the README if you want to find out how to configure this stuff.

  • Manually set window opacity using bindable callbacks
  • Automatically set window opacity using window matching rules
  • Control a MPD instance using bindable callbacks
  • Inform about the player state (song info, player settings) of a MPD instance using desktop notifications either when triggering a bindable callback or on player state changes (uses a separate watch thread)
  • Bookmark time positions within songs and playlists
  • wmii-like actions
  • Compile with -O3 -march=native
